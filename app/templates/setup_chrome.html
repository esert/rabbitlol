How do I add rabbitlol as a search engine to Chrome?

Chrome adds rabbitlol as a search engine when you visit www.rabbitlol.com. You can verify it by going to Settings > Manage search engines (under Search title) and then searching for rabbitlol under Other search engines.
If you don't see rabbitlol command in Chrome at the described location, you can add it manually. At the very end of Other search engines list, you will see a data entry row. Set the name to rabbitlol, keyword to rabbitlol.com and URL to 
https://www.rabbitlol.com/?q=%s
You can run rabbitlol commands by typing rabbitlol.com to omnibox then hitting Tab button.

How do I make rabbitlol the default search engine in Chrome?

Go to Settings > Manage search engines and find rabbitlol under Other search engines. Hover your mouse over rabbitlol and you should see Make default appear at the right end of the row. Clicking it will make rabbitlol your default search engine, so you can query rabbitlol from your omnibox with no extra effort.

How do I set a default command?

If you are signed in, visit commands page and set one of your commands as the default command.
If you are not signed in, you need to tell rabbitlol which command to use as default command everytime you make a query. To do that, you need to modify the search URL. Go to Settings > Manage search engines, find rabbitlol and add &default=<your_defualt_command_name> at the end of the query URL. For example, if you want to use g as your default command, convert 
https://www.rabbitlol.com/?q=%s
 to 
https://www.rabbitlol.com/?q=%s&default=g
