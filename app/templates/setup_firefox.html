How do I add rabbitlol as a search engine to Firefox?

Click the magnifier next to the search bar. On the pane that just appeared, click Add rabbitlol. You will be able to choose rabbitlol as your search engine next time you use your search bar.

How do I make rabbitlol the default search engine in Firefox?

Go to Preferences > Search. Select rabbitlol from the dropdown menu under the Default Search Engine title. rabbitlol is now your deafult search engine.

How to set a default command?

If you are signed in, visit commands page and set one of your commands as the default command.
If you are not signed in, there isn't a way to provide a default command in Firefox.
